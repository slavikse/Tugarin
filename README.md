# Tugarin

## Команды
* Запуск: `npm run serve`
* Сборка: `npm run build`
* Путь: `out/make/squirrel.windows/x64`

## Управление
* **WASD** - Перемещение.
* **Arrows** - Выстрелы.
* **Shift** - Замедление скорости.
* **F1** - Отобразить время кадра.

## Система координат блока относительно игрока.
```
        Y ^
          |
          |
X         |
<---------|
```
